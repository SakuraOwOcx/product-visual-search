# Model and Data Files

This repository intentionally does not track large local artifacts such as raw datasets, model checkpoints, CLIP weights, or embedding indexes.

## Dataset

Place the dataset files under:

```text
data/raw/
```

Expected important files:

```text
data/raw/images/
data/raw/styles.csv
```

The project uses `data/raw/images/` as the canonical image folder and avoids duplicate extracted image folders such as `data/raw/myntradataset/images/`.

## CLIP Checkpoint

The local CLIP checkpoint used by the demo is expected at:

```text
models/huggingface/timm_vit_base_patch32_clip_224_openai/open_clip_model.safetensors
```

This file is too large for normal GitHub tracking. Use the existing download script if needed:

```powershell
python .\_process_archive\download_clip_weights.py
```

or place the checkpoint manually at the path above.

## ResNet18 Full Checkpoint

The full ResNet18 checkpoint is generated by:

```powershell
python .\scripts\train_resnet18_full.py --epochs 10 --batch-size 64 --num-workers 4
```

Expected output:

```text
outputs/checkpoints/resnet18_full_best.pth
```

## Embedding Indexes

The Streamlit demo uses cached vector indexes:

```text
outputs/indexes/clip_full_gallery_index.npz
outputs/indexes/resnet18_full_gallery_index.npz
```

They can be rebuilt with:

```powershell
python .\scripts\build_clip_full_index.py
python .\scripts\build_resnet18_full_index.py
```

These indexes are excluded from Git because they are generated binary artifacts.
